# 初始化云项目配置
主要做一个适合自己的小程序云开发公共模板项目，方便自己创建进行开发。做了个简单的页面DEMO！主要使用组件是lin-ui...

# 项目简单介绍
 - 使用的组件是[lin-ui](https://doc.mini.talelin.com)组件；
 - anpai-ui是基于lin-ui进一步封装，作者自定义的ui库;
 - 当前项目会持续更新，让开发更加便捷！
